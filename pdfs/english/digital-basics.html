Digital Basics Guide

Introduction to Computers

Basic Computer Components

  • Monitor - The screen that displays information
  • Keyboard - Used for typing and input
  • Mouse - Used for navigation and clicking
  • CPU (Central Processing Unit) - The brain of the computer
  • Storage - Where files and programs are saved

Starting and Shutting Down

  1. Press the power button to start
  2. Wait for the computer to boot up
  3. Log in with your username and password
  4. To shut down: Click Start → Power → Shut down

Using the Internet

Web Browsers

Popular web browsers include:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

Basic Internet Navigation

  1. Open your web browser
  2. Type a website address in the address bar
  3. Use the back and forward buttons to navigate
  4. Click on links to visit different pages
Tip: Always verify website addresses before entering personal information.

Email Basics

Creating an Email Account

  1. Choose an email provider (Gmail, Outlook, etc.)
  2. Click on "Create Account" or "Sign Up"
  3. Fill in your personal information
  4. Choose a strong password
  5. Verify your account

Sending and Receiving Emails

  • Click "Compose" to write a new email
  • Enter recipient's email address
  • Write your message
  • Click "Send"
  • Check your inbox for new messages

File Management

Creating and Organizing Files

  1. Right-click → New → Folder to create a new folder
  2. Name your folder appropriately
  3. Drag and drop files into folders
  4. Use descriptive names for files

Basic File Operations

  • Copy: Ctrl+C
  • Paste: Ctrl+V
  • Cut: Ctrl+X
  • Delete: Delete key
  • Undo: Ctrl+Z

Online Safety

Creating Strong Passwords

  • Use at least 8 characters
  • Include numbers, letters, and symbols
  • Avoid using personal information
  • Use different passwords for different accounts
Important: Never share your passwords with anyone and be cautious of suspicious emails or links.
